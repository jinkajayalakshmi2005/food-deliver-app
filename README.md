Food Delivery Web Application
# Project Description
This project is a simple Food Delivery Web Application developed using HTML, CSS, and JavaScript. The application allows users to view different food items with images and prices, add items to a cart, and place orders through a basic payment page.

This project demonstrates fundamental front-end web development concepts, including user interface design, cart functionality, and simple order processing.
# Features

Display food items with images and prices

Add to Cart functionality

Cart page to view selected items

Payment page to place the order

Simple and clean user interface

# Technologies Used

HTML – Structure of the web pages

CSS – Styling and layout design

JavaScript – Cart functionality and order processing
